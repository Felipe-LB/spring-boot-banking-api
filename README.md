# Calculadora feita em java# 💰 Spring Boot Banking API

RESTful API for banking operations built with Spring Boot (accounts, transactions, and balance management).

---

## 🎯 Objective

This project aims to implement a backend banking system with basic operations such as account creation, deposits, withdrawals, and transfers.

---

## 🛠️ Tech Stack

* Java 17
* Spring Boot
* Spring Web
* Bean Validation
* Maven

---

## 📌 Planned Features

* [ ] Create bank accounts
* [ ] Deposit and withdraw money
* [ ] Transfer between accounts
* [ ] Balance inquiry
